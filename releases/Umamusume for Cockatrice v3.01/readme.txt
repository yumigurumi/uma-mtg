To install:
-Download and unzip file
-Empty contents of "Cockatrice" folder into C:\Users[YOUR USERNAME]\AppData\Local\Cockatrice\Cockatrice\
-It should ask to merge contents of folders into the "customsets" folder (this is for card data) and "pics" (this is for card images).
-Relaunch Cockatrice OR click Card Database > Reload Card Database
-YAY!!!!!!! (Check that the set is enabled in Card Database > Manage Sets, and that you can access and view the cards with images.)
