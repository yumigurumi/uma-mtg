# Umamusume: Universes Beyond - UUMA
This is a Cockatrice-compatible Umamusume-inspired Universes Beyond set for Magic the Gathering.

## Cockatrice installation (for Windows):
- Go to the [Releases](https://github.com/yumigurumi/uma-mtg/releases) page.
- Download the latest release.
- Unzip.
- Empty the contents of the "Cockatrice" folder into `C:\Users[YOUR USERNAME]\AppData\Local\Cockatrice\Cockatrice\`.
- Relaunch Cockatrice OR click Card Database > Reload Card Database
- Enjoy! (Check that the set is enabled in Card Database > Manage Sets, and that you can access and view the cards with images.)
